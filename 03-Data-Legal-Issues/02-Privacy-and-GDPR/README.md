# AI 시스템의 프라이버시 및 GDPR 준수 지침

## 개요

본 문서는 AI 시스템 개발 및 운영 시 준수해야 할 프라이버시 보호 원칙과 GDPR 요구사항을 제시합니다. 특히 AI 시스템의 특성을 고려한 구체적인 이행 방안과 기술적 조치를 포함합니다.

## 1. GDPR 핵심 원칙

### 적법성, 공정성, 투명성
- **적법한 처리 근거**
  - 명시적 동의
  - 계약 이행
  - 법적 의무
  - 정당한 이익

- **공정한 처리**
  - 차별 방지
  - 편향성 제거
  - 균형있는 결과

- **투명한 처리**
  - 알고리즘 설명
  - 의사결정 로직 공개
  - 영향 고지

### 목적 제한 및 데이터 최소화
- **목적 명확화**
  - 구체적 목적 정의
  - 목적 외 사용 제한
  - 목적 변경 절차

- **최소 데이터**
  - 필요 데이터 정의
  - 불필요 데이터 제거
  - 저장 기간 설정

### 자동화된 의사결정

  - 영향 평가

### 설명가능성 요구사항

  - 로직 설명

  - 영향 설명

  - LIME 활용
  - SHAP 구현
  - 모델 카드 작성
